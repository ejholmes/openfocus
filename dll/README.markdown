## Hypnofocus DLL
DLL used to communicate with the firmware. Provides the following functions:
`focuser_connect`
`focuser_disconnect`
`focuser_move_to`
`focuser_halt`
`focuser_is_moving`
`focuser_get_position`
 
### Requirements
* MinGW

### Building from Source
1. run `make`
